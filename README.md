# AT3112701-Robotics-Sand-Machine

## Devices 
### MCU
 - ESP32 (Node32S)
### Stepper Motors
 - 42BYGH34 * 2
 - FOYO TB6600 Digital Step Driver * 2
### Sensors
 - PM-R25
 - EZ-Q2N
## Software
Written with Arduino Framework.  
With a LabVIEW Control Station.
